# zero-cloud
SpringCloud 微服务开发框架，基于 Spring Cloud Finchley 、Spring Security OAuth2 的简易RBAC权限管理系统
### 模块说明
```bash
api-common -- 通用工具类
api-model -- 通用实体类
config-server -- 配置中心
eureka-server -- 服务注册中心
gateway-zuul -- 服务网关
log-server -- 日志中心
log-starter -- 日志依赖starter
monitor-server -- 监控中心
oauth-server -- 认证中心
user-server -- 用户中心
zero-cloud-web -- 前端文件
```
